Câu hỏi 20Đã bỏ qua

Which of the following results in faster access to internal tables? (3 correct)

In a hashed internal table, specifying the primary key partially from the left without gaps.

Lựa chọn đúng

In a sorted internal table, specifying the primary key partially from the left without gaps.

Lựa chọn đúng

In a hashed internal table, specifying the primary key completely.

Lựa chọn đúng

In a sorted internal table, specifying the primary key completely.

In a standard internal table, specifying the primary key partially from the left without gaps.

For hashed internal tables, specifying the primary key partially from the left without gaps is false. In a hashed internal table, the key must be complete and without gaps for optimal performance.

Hashed tables use a hash algorithm to calculate a hash code based on the key fields, and this code is used to determine the storage location of the data in the table. If the primary key is specified partially or with gaps, it can lead to hash collisions, where different entries end up with the same hash code. This can result in degraded performance and defeat the purpose of using a hashed table.

To ensure fast and efficient access in a hashed internal table, you should specify the complete primary key without any gaps.

Câu hỏi 42Đã bỏ qua

Which restrictions exist for ABAP SQL arithmetic expressions? (2 correct)

Lựa chọn đúng

The operator / is allowed only in floating point expressions.

The operator * is allowed only in floating point expressions.

Lựa chọn đúng

Floating point types and integer types can NOT be used in the same expression.

Decimal types and integer types can NOT be used in the same expression.

In ABAP SQL arithmetic expressions, certain restrictions exist to maintain data type consistency and avoid potential errors or inconsistencies in calculations. These restrictions are in place to ensure that arithmetic operations are performed correctly and efficiently. Here's why each of the statements is true:

Floating point types and integer types can NOT be used in the same expression: Mixing floating point and integer types in the same expression can lead to unexpected results, as the two types handle decimal points differently. Floating point types allow for decimal fractions, while integer types do not. Combining them in the same expression can result in implicit conversions and loss of precision.

The operator '/' is allowed only in floating point expressions: Division involving floating point and integer types can lead to loss of precision or unexpected results. Therefore, the division operator '/' is restricted to floating point expressions to ensure that division operations are performed with floating point precision.

Tùy chọn cookie

Chúng tôi sử dụng cookie để lưu trữ hoặc truy xuất thông tin trên trình duyệt của bạn. Thông tin này có thể liên quan đến bạn, sở thích của bạn hoặc thiết bị của bạn và chủ yếu được sử dụng để làm cho trang web hoạt động như bạn mong đợi. Vì chúng tôi tôn trọng quyền riêng tư của bạn nên bạn có thể chọn không cho phép một số loại cookie. Nhấp vào các tiêu đề thể loại khác nhau để tìm hiểu thêm và thay đổi các cài đặt của bạn. Tuy nhiên, việc chặn một số loại cookie có thể ảnh hưởng đến trải nghiệm của bạn về trang web và các dịch vụ mà chúng tôi có thể cung cấp.

Cookie thiết yếu

Luôn hoạt động

Những cookie này cần thiết để trang web hoạt động và không thể tắt trong hệ thống của chúng tôi. Chúng thường chỉ được cài đặt để đáp ứng các hành động do bạn thực hiện tương ứng với yêu cầu dịch vụ, chẳng hạn như cài đặt tùy chọn quyền riêng tư của bạn, đăng nhập hoặc điền vào biểu mẫu. Bạn có thể cài đặt trình duyệt của bạn để chặn hoặc thông báo cho bạn về những cookie này, nhưng sau đó, một số phần của trang web sẽ không hoạt động. Những cookie này không lưu trữ bất kỳ thông tin nhận dạng cá nhân nào.

Cookie chức năng

Không hoạt động

Những cookie này cho phép trang web cung cấp chức năng nâng cao và cá nhân hóa. Chúng có thể do chúng tôi cài đặt hoặc do các nhà cung cấp bên thứ ba có dịch vụ mà chúng tôi đã thêm vào các trang của chúng tôi cài đặt. Nếu bạn không cho phép những cookie này thì một số hoặc tất cả các dịch vụ này có thể không hoạt động bình thường.

Cookie hiệu suất

Không hoạt động

Những cookie này cho phép chúng tôi đếm lượt truy cập và đo lường hoạt động để chúng tôi có thể cải thiện hiệu suất của trang web của chúng tôi. Chúng giúp chúng tôi biết được trang web nào phổ biến nhất và ít phổ biến nhất và xem cách người dùng di chuyển trên trang web. Nếu bạn không cho phép những cookie này, chúng tôi sẽ không thể theo dõi hoạt động của cookie.
